## WHAT IS AN ANNUAL RETURN?

Every company registered with the CIPC must file an Annual Return each year within a set filing window. This is a statutory obligation — failure results in deregistration.

> ⚠️ **Annual Returns are NOT the same as SARS tax returns.** These are filed separately with CIPC.

## PRE-FILING CHECKLIST

### Company Information Verification

- [ ] Company name is correct and unchanged (or name change has been filed — COR9.5)
- [ ] Registered address is current (or address change filed — COR21.1)
- [ ] All directors are correctly listed (no undisclosed changes — COR39)
- [ ] Company type is still correct (Pty Ltd / Inc. / etc.)

### Director Changes (file these before annual return if applicable)

- [ ] New director appointments: **COR39** — must be filed within **10 business days** of appointment
- [ ] Director resignations: **COR40** — filed within 10 business days
- [ ] Director changes (address, ID update): **COR39.3**

## FILING PROCESS (BizPortal / CIPC eFiling)

1. Log in to **www.bizportal.gov.za** with CIPC account credentials
2. Select **Annual Returns > File Now**
3. Confirm company details (auto-populated from CIPC records)
4. Confirm turnover for the year (must match financial period)
5. Upload financial statements (if required)
6. Calculate and pay the annual return fee (EFT / credit card)
7. Download and save the **filing confirmation / COR30.1A** certificate
8. File in company records

## BENEFICIAL OWNERSHIP REGISTER (BOD)

As of **1 April 2023**, all companies must maintain and submit a **Beneficial Ownership Register** to CIPC.

**Beneficial Owner Definition:** Any natural person who ultimately owns or controls ≥ 5% of shares, or exercises effective control.
