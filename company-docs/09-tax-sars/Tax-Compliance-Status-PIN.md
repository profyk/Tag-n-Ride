## WHAT IS A TCS PIN?

A Tax Compliance Status (TCS) PIN is a unique reference number issued by SARS that allows a third party (e.g., bank, investor, government tender body, regulator) to verify that Tag-n-Ride is tax compliant **without** disclosing the company's confidential tax affairs.

---

## WHEN IS A TCS PIN REQUIRED?

| Scenario | Requirement |
|----------|-------------|
| Applying for a bank account or banking facility | Yes |
| Government tender / RFP submission | Yes (Tender PIN) |
| Applying for FSCA licence or exemption | Likely |
| Investor due diligence | Yes |
| Foreign investment > R10,000,000 | Yes (Emigration/Foreign Investment PIN) |
| Proof of good standing for commercial contracts | Yes |
| CIPC annual return | Good standing recommended |

## PREREQUISITES FOR ISSUANCE

SARS will only issue a Good Standing PIN if:
- [ ] All tax returns are submitted and up to date (CIT, VAT201, EMP201, EMP501)
- [ ] No outstanding tax debt (or a formal arrangement/deferment is in place)
- [ ] No active audit objection blocking compliance status

---

## HOW TO OBTAIN A TCS PIN

### Via SARS eFiling (Preferred)
1. Log in to eFiling: www.sarsefiling.co.za
2. Navigate to **Home > Tax Status > Request Tax Compliance Status**
3. Select **"Good Standing"**
4. Confirm all returns and payments are up to date
5. Submit request
6. TCS PIN generated immediately (valid for **1 year** from issuance)
7. Share PIN with requesting third party — they verify at sars.gov.za/verify

### Via SARS MobiApp
1. Download SARS MobiApp
2. Log in with eFiling credentials
3. Select **Tax Compliance Status > Request PIN**

## REMEDYING NON-COMPLIANCE

If SARS has blocked the TCS PIN due to:

| Issue | Action |
|-------|--------|
| Outstanding returns | Submit overdue returns immediately via eFiling |
| Outstanding debt | Apply for a **Payment Arrangement** (PA) via eFiling |
| Audit in progress | Contact SARS to request **"Audited but Compliant"** status |
| Disputed assessment | Lodge formal objection (ADR1) within 30 days of assessment |
